# Spring MVC CRUD Application (JSP)

## Description
Simple Spring MVC CRUD web application using JSP as view technology.  
The application allows users to create, view, and manage product records.

## Technologies Used
- Java
- Spring MVC
- JSP
- JSTL
- Maven
- HTML, CSS
- MySQL (or any relational database)
- Apache Tomcat

## Features
- Add Product
- View All Products
- View Product Details
- Update Product
- Delete Product

## Project Structure
src/main/java
- controller
- service
- repository (dao)
- model

src/main/webapp
- WEB-INF/views (JSP files)

pom.xml

## Setup Steps
1. Clone repository
git clone https://github.com/srihari9398/SpringMVC_CRUD_App.git

2. Import project as Maven project in STS / Eclipse / IntelliJ.

3. Configure database connection.

4. Build project
mvn clean install

5. Deploy on Apache Tomcat server.
